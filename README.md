
# 🔍 Crime Analysis System

Machine Learning system for detecting crime tools and analyzing crime data.

## Features

- 📊 **Text Analysis**: Extract and analyze crime-related text
- 🤖 **ML Models**: Train multiple models (SVM, Random Forest, Neural Networks)
- 🔮 **Tool Detection**: Predict crime tools from text descriptions
- 📈 **Performance Metrics**: Compare model accuracy, precision, recall, F1-score
- 🎨 **Interactive Dashboard**: Filter and explore crime data with Gradio
